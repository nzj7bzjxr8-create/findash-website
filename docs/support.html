Frequently Asked Questions

How do I import my broker's CSV file?

See the User Guide section on Importing Data for step-by-step instructions for your broker.

Is my data secure?

Yes. All data is stored locally on your device. See our Privacy Policy for security details.

Can I sync my portfolio across devices?

iCloud sync is available on macOS. iPad support is planned for a future release — once available, sync will extend to iPad as well.

Which brokers are supported?

FinDash can import CSV files from any broker. See the User Guide for broker-specific instructions.

How do I get live market data?

Configure optional API keys in Settings. Supported providers: Alpha Vantage, Tiingo, Massive, and FMP. See the User Guide for setup instructions.

What if I find a bug?

Please contact us using the form above or email info@dashwellsolutions.com. Include details about what you were doing and what happened.
